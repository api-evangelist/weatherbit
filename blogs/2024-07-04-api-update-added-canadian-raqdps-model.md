<img alt="API Update: Added Canadian RAQDPS Model" src="https://blog.weatherbit.io/content/images/2024/07/2024070400_054_GM_north@america@gemmach_I_GEMMACH_o3@sfc@006_006.png" /><p></p><p>We have added <a href="https://weather.gc.ca/aqfm/index_e.html?ref=blog.weatherbit.io" rel="noreferrer">RAQDPS</a> model data to our <a href="https://www.weatherbit.io/api/airquality-forecast?ref=blog.weatherbit.io" rel="noreferrer">Air Quality Forecast API</a> which greatly improves air quality forecasts in North America. This model provides air quality forecasts out to 72 hours, and updates twice per day.</p><p>There are no integration changes necessary to use this data. Any air quality forecast request within the CONUS/Canada region will use data from this model to form it&apos;s forecast.</p><h2 id="conclusion">Conclusion</h2><p>We hope that our users find value in these updates. We are always looking for ways to improve our API. Feel free to reach out to support@weatherbit.io with any questions or concerns.</p>
