<img alt="API Update: Global Lightning API" src="https://blog.weatherbit.io/content/images/2024/08/NOAA-Lightning-Strike-FInal.webp" /><p>We are excited to announce the release of our Global <a href="https://www.weatherbit.io/api/lightning-current?ref=blog.weatherbit.io" rel="noreferrer">Lightning API</a>! This API allows for the retrieval of lightning flash data for any location in the world. Data is sourced from geostationary satellite lightning sensors, as well as weather radar. </p><p>By default it will return all lightning  flashes that occurred within the last 30 minutes, and 50 kilometers from the point. Example:</p><pre><code class="language-js">https://api.weatherbit.io/v2.0/current/lightning?lat=48.1351&amp;lon=11.5820&amp;key=API_KEY</code></pre><h5 id="alternatively-you-can-supply-any-of-the-following-parameters">Alternatively, you can supply any of the following parameters: </h5><ul><li><strong>limit [Limit the total number of results returned]</strong></li><li><strong>search_dist_km [Search distance radially from provided point]</strong></li><li><strong>search_mins [How many minutes backwards from current time to search for lightning data]</strong></li><li><strong>sort [Sort results either by distance or time]</strong></li></ul><p>So if you would like to retrieve all lightning for the past hour, within 100 KM of a point (48.1351, 11.5820) sorted by most recent time the request would simply be:</p><pre><code class="language-js">https://api.weatherbit.io/v2.0/current/lightning?lat=48.1351&amp;lon=11.5820&amp;search_dist_km=100&amp;search_mins=60&amp;sort=time&amp;key=API_KEY</code></pre><p>This new API is available now to all currently offered commercial subscriptions. </p><h2 id="conclusion">Conclusion</h2><p>We hope that our users find value in these updates. We are always looking for ways to improve our API. Feel free to reach out to support@weatherbit.io with any questions or concerns.</p>
